'Numpy'

numpy的使用

1.使用np的routines函数创建数组

  • 所有元素为1的数组

    np.ones(shape, dtype=None, order='C')

  • 所有元素为0的数组

    np.zeros(shape, dtype=float, order='C')

  • 任意元素的数组（所有元素相同）

    np.full(shape, fill_value, dtype=None, order='C')

  • 对角线为1的数组，对角线可移动

    np.eye(N, M=None, k=0, dtype=float)

  • 等差数列数组

    np.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None)

  • 等距数组

    np.arange([start, ]stop, [step, ]dtype=None)

  • 随机整数数组

    np.random.randint(low, high=None, size=None, dtype='l')

  • 标准正太分布数组/自定义正太分布数组

    np.random.randn(d0, d1, ..., dn) np.random.normal(loc=170,scale=20,size=(10,3))

  • 随机小数数组（0-1）

    np.random.random(size=None)

2.ndarray属性

ndim:维度，shape:形状，size:长度，dtype:元素类型

3.ndarray基本操作

  • 索引

    • 一维单元素索引

      arr[i]

    • 一维多元素索引

      arr[[m,n,p]]

    • 多维索引

      arr[m,n,p]

  • 切片

    • 一维切片（左开右闭）

      arr[m:n]

    • 一维等距切片

      arr[::n]

    • 高维切片

      • 行切

        arr[n:m]

      • 列切

        arr[:,n:m]

    • 图片翻转

      • 上下/左右

        img_data[::-1]/img_data[:,::-1]

      • 三维

        img_data[:,:,::-1]

  • 变形

    • reshape函数，参数是tuple

      • 变形时注意前后两者size必须相同

  • 级联

    • np.concatenate()

      • 级联的参数是列表
      • 维度相同
      • 形状相符
      • 【重点】级联的方向默认是shape这个tuple的第一个值所代表的维度方向
      • 可通过axis参数改变级联的方向

      np.concatenate((arr1, arr2, arr3),axis=0)

    • np.hstack与np.vstack 水平级联与垂直级联,处理自己，进行维度的变更

      np.hstack((arr1, arr2))

  • 切分

    • np.split/np.vsplit/np.hsplit

      np.split(arr1, indices_or_sections=2, axis=1)

      np.split(arr1, indices_or_sections=[2,3], axis=1)

      np.hsplit(arr1, indices_or_sections=[2,3])

  • 副本

    • 所有赋值运算不会为ndarray的任何元素创建副本。对赋值后的对象的操作也对原来的对象生效。
    • 可使用copy()函数创建副本
    • 深浅拷贝

4.ndarray聚合操作

  • 求和np.sum

    arr.sum(axis=(0,1,2)) arr.sum(axis=1) arr1.sum()

  • 最大最小值：np.max/ np.min

    arr.max(axis=0)

    • 求最大值和最小值的索引

      arr.argmax() arr.argmin()

  • 计算所有元素的乘积,标准差,平均值

    arr.prod() arr.std() arr.mean()

  • arr.any()/arr.all()

    • 当前数据集中，是否至少存在一个True

    • 当前数据集中，是否全部都是True

    • numpy.array支持使用一个等长的bool列表来做索引提取True对应的值

  • 使用数据集计算出一个bool型条件列表， 所有满足条件的都记录为True；然后再用这个bool型的列表来提取原始数据；可以使用any函数，来确认条件列表中是否存在满足条件的项

  • np.sum 和 np.nansum 的区别：nan not a number

    arr = np.array([1,2,3,4,np.nan])

    • 面向对象的方法

      arr.sum()——nan

    • 面向过程式的函数

      np.nansum(arr)——10

5.ndarray矩阵操作

  • 基本矩阵操作
    • 加减乘除
    • 矩阵积np.dot()
  • 广播机制
    • ndarray广播机制的两条规则
      • 规则一：为缺失的维度补1
      • 规则二：假定缺失元素用已有值填充
    • 一条原则，任何两个形状不同的数组之间运算，都要先完成形状相同的变形操作，再运算
    • 例
      • a = np.arange(3).reshape((3, 1)) b = np.arange(3) 求a+b
      • a = np.ones((4, 1)) b = np.arange(4) 求a+b

6.ndarray的排序

  • np.sort()与ndarray.sort()都可以

    arr = np.array([1,5,7,9,8,3,2,6,4,0])

    • np.sort()不改变输入

      arr1 = np.sort(arr)arr1改变，arr不变

    • ndarray.sort()本地处理，不占用空间，但改变输入

      arr1.sort()arr和arr1都改变

  • 部分排序 np.partition(a,k)

    • 当k为正时，我们想要得到最小的k个数

      np.partition(arr, 2)

    • 当k为负时，我们想要得到最大的k个数

      np.partition(arr, -2)

Series

  • Series是一种类似于一维数组的对象，由下面两个部分组成：

    • values：一组数据（ndarray类型）
    • index：相关的数据索引标签

  • series对象如果没有显示指定index，则把数组的索引设置成index

    • 使用index参数显示设置索引

      s2 = Series(list1, index=['name','python','c','java','php','address'])

  • series接受的data必须是一维数组

Series创建三种方法

  • 使用列表构建，副本对象，副本对象相互独立

  • 使用数组构建，使用numpy构建的是引用对象，引用对象会相互影响

  • 使用字典构建

Series的索引和切片

  • 索引

    • 可以使用中括号取单个索引（此时返回的是元素类型），或者中括号里一个列表取多个索引s1[["语文","数学"]]（此时返回的仍然是一个Series类型）。分为显示索引和隐式索引

    • 显示索引

      • 使用index中的元素作为索引值

      • 使用.loc[] (推荐)

        s1.loc["语文"]

    • 隐式索引

      • 使用整数作为索引值

      • 使用.iloc[] (推荐)

        s1.iloc[[0,2]]

  • 切片

    s2 = Series(data=np.random.randint(0,100,size=10), index=list("ABCDEFGHIJ"))

    • 使用显式索引切片 左闭右闭区间

      s2.loc["B":"J"]

    • 使用隐式索引切片 左闭右开区间

      s2.iloc[0:3]

Series的基本概念

  • 可以把Series看成一个定长的有序字典

  • 可以通过shape，size，index，values等得到series的属性

  • 可以使用head()，tail()分别查看前n个和后n个值

  • 当索引没有对应的值时，可能出现缺失数据显示NaN（not a number）的情况

  • 可以使用pd.isnull()，pd.notnull()，或自带isnull()，notnull()函数检测缺失数据

    • 可以使用notnull函数过滤空值

  • 使用Bool_list访问数组对象

  • Series对象本身及其实例都有一个name属性

Series的运算

  • Series运算与ndarray运算规则的区别

  • 一条原则，索引对齐；跟数字运算，不会出现索引对齐的问题

  • 适用于numpy的数组运算也适用于Series

  • Series之间的运算

    • 在运算中自动对齐不同索引的数据
    • 如果索引不对应，则补NaN

  • 注意：要想保留所有的index，则需要使用

    • add() 加

    • sub() 减

    • mul() 乘

    • div() 除

      指定fill_value参数，当需要使用空值填充时，把空值替换成fill_value的值
